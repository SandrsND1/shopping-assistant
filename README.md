# 🛒 Shopping Assistant

[![Python Version](https://img.shields.io/badge/python-3.8%2B-blue)](https://python.org)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)
[![GitHub stars](https://img.shields.io/github/stars/твой-логин/shopping-assistant)](https://github.com/твой-логин/shopping-assistant/stargazers)

Консольное приложение для управления покупками с категориями товаров, списками и аналитикой расходов. Помогает планировать покупки и отслеживать траты.

---

## ✨ Возможности

- 📁 **Управление категориями** — создавайте, редактируйте и удаляйте категории товаров
- 📦 **Управление товарами** — добавляйте товары с ценами и единицами измерения
- 📋 **Списки покупок** — создавайте списки, добавляйте товары, отмечайте купленное
- 📊 **Статистика** — анализируйте расходы по категориям, смотрите топ товаров
- 💾 **База данных** — всё сохраняется в SQLite, данные не теряются после выхода

---

## 🚀 Установка и запуск

### Требования
- Python 3.8 или выше
- Git (для клонирования)

### Шаги
```bash
# Клонируй репозиторий
git clone https://github.com/твой-логин/shopping-assistant.git

# Перейди в папку проекта
cd shopping-assistant

# Запусти приложение
python main.py

Запушь: git push origin feature-name

Открой Pull Request
```
